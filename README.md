# norobot
Simple graphic CAPTCHA that shows actual words instead of random set to help a human read it.

Usage:
```
>>> import norobot
>>> norobot.gen_captcha()
{'image': <PIL.Image.Image image mode=RGB size=280x60 at 0x7F64A900E898>, 'word': 'мораль'}
```
